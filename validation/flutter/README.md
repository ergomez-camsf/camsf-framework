# Flutter

Esta carpeta contiene la implementación de referencia desarrollada en Flutter utilizada para la validación experimental del framework CAMS-F.

Los artefactos incluidos ilustran la integración del enfoque basado en modelos con Flutter y Firebase, permitiendo validar la generación de aplicaciones móviles sensibles al contexto.

Archivos incluidos:

- `main.dart`
- `auth_repository.dart`
- `firestore_repository.dart`
- `firestore.rules`
