# DSL – CAMS-F

Esta carpeta contiene la definición textual del lenguaje específico de dominio (DSL) desarrollado mediante Xtext para el framework CAMS-F.

## Archivos principales

- **cams.xtext**: versión principal del DSL utilizada para el modelado de aplicaciones móviles sensibles al contexto.

- **camsdsl_grammar.xtext**: gramática completa del DSL integrada con el metamodelo EMF y utilizada durante el proceso de desarrollo del lenguaje.

## Descripción

El DSL permite modelar los principales conceptos del dominio de las aplicaciones móviles sensibles al contexto, incluyendo objetos conscientes del contexto (Aware Objects), características contextuales, sensores, servicios y reglas de adaptación definidas en el framework CAMS-F.
